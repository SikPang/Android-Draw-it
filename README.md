### Creativity enhancing minigames for Android (with. 주세현)
  <br/>
  <div align="left">
  Android Studio(Java, XML)를 이용한 Android용 창의력 강화 미니게임 어플리케이션 제작 2인 팀 프로젝트 "그려보아요" 입니다. <br/>
저연령 대상으로 미니게임의 메인 오브젝트를 유저가 직접 그려서 플레이할 수 있도록 했고, <br/>
미니게임은 총 4가지로 사과 받기, 비행기 슈팅, 점프, 모험 게임 입니다. <br/>
게임의 기본적인 틀, 메인 아이템, 레벨 디자인 등의 전체적인 기획을 맡았고, 미니게임 위주의 개발을 진행하였습니다.<br/><br/>
- 회원 가입, 로그인 및 회원 데이터 기기 내에 저장<br/>
- Rect, intersects 라이브러리를 이용한 오브젝트 간 충돌 처리<br/>
- 터치하는 곳으로 캐릭터를 이동시키는 게 아닌, 터치하는 위치와 상관 없이 손가락을 움직이는 만큼만 캐릭터를 움직일 수 있게끔 onTouchEvent 설계<br/>
- Android Studio에 내장되어 있는 SQLite를 이용해 데이터베이스 작업을 하여 같은 디바이스 내에서의 랭킹 시스템 구현<br/>
